# Proyecto-Algoritmo-de-Busqueda-de-contrasenas-por-diccionario
Repositorio del proyecto de la materia Analisis de Algoritmos, el proyecto consiste en un algoritmo de busqueda de contrasenas por diccionario utilizando distintas tecnicas vistas a lo largo del semestre  

El algoritmo esta elaborado con la finalidad de comprobar la seguridad de las contraseñas de los usuarios probando su integridad por medio de un ataque simulado de diccionario. Principalmente, buscamos darle un enfoque educativo al proyecto, planteando la implementacion de tecnicas de elaboracion de algoritmos y observar su funcionamiento dentro de un algoritmo como este. 

### Funcionamiento del programa 

El programa cuenta con una interfaz realizada con Tkinter, donde el usuario puede:
-Establecer una contraseña objetivo
-Elegir metodo de busqueda
-Iniciar/detener la ejecucion del programa
-Visualizar el registro de intentos 
-Utilizar entre distintos diccionarios de contraseñas


### Dependencias 

Se recomienda utilizar Python 3.10 y versiones posteriores, debido a que ya cuentan con las librerias necesarias incluidas desde su instalación. En ocaciones se requiere la instalacion remota de la libreria matplotlib, para ello puede usarse el comando pip 

```bash
pip install matplotlib
```


### Instrucciones de ejecución

-Descargar el archivo Busqueda_contrasenas_huffman.py \
-Utilizar un IDE para abrir el archivo y ejecutarlo o bien ejecutarlo desde consola 


### Conclusiones 
-El ataque por diccionario es una tecnica eficiente y rapida cuando la contraseña esta presente dentro del diccionario
-Un ataque de fuerza bruta por combinaciones es exponencial y escala mal conforme aumenta la longitud de la contraseña
-Divide y venceras ayuda al manejo y ataque de varias contraseñas a la vez, resolviendo problema con maypr velocidad 
-El metodo voraz con Huffman permite ahorrar espacio en el almacenamiento de los diccionarios
