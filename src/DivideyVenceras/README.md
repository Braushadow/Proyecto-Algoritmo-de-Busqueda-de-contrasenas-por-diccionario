Tecnica de Divide y Vencerás 

La tecnica de divide y vencerás en nuestro algoritmo es utilizada desde distintos enfoques, uno de ellos es que el algoritmo ahora acepta varias contraseñas en lugar de solo una
de modo que el algoritmo tiene mayor posibilidad de averiguar varias contraseñas a la vez. 

Ademas, el algoritmo tiene una funcion de worker con hilos, de modo que se crea un hilo por cada contraseña ingresada para averiguar y de esta manera se realizan 
distintos intentos para averiguar contraseñas de manera paralela sin necesidad de tener que hacer todos los intentos de manera serial, lo que reduce el tiempo de ejecucion del
programa ofreciendo mejores resultados en menor tiempo.
